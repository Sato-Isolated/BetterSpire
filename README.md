# BetterSpire2 Lite

BetterSpire2 Lite is a Slay the Spire 2 quality-of-life mod focused on clearer combat information, better multiplayer visibility, and a few convenience upgrades.

## Features

### Combat HUD improvements
- Shows total damage for multi-hit enemy intents.
- Shows total incoming damage above the player.
- Optionally shows expected HP after incoming damage resolves.
- Tracks projected pet damage when relevant.
- Adds a movable, resizable turn summary tracker for combat and round-by-round review.
- Turn summary data includes cards played, damage dealt and taken, block gained and lost, energy flow, and per-card breakdowns.

### Multiplayer tools
- Adds a teammate hand viewer that can be toggled during combat.
- Supports auto-opening the hand viewer when combat starts.
- Can keep the hand viewer open, hide your own hand, or switch to a compact stats-only view.
- Lets you adjust hand viewer card size.
- Supports paging through players when the party is large.
- Adds party controls in the settings menu to hide specific teammates' map drawings or clear all drawings.

### Quality-of-life additions
- Adds an in-game settings menu.
- Supports optional ModConfig integration when that mod is available.
- Adds a draggable clock overlay with optional 24-hour time.
- Adds a combat-only instant fast mode option.
- Adds an option to skip the splash screen.
- Saves UI layout and settings between sessions.

## Hotkeys
- `F1` - Open or close the BetterSpire2 Lite settings menu.
- `F3` - Open or close the teammate hand viewer.
- `Page Up` / `Page Down` - Change pages in the teammate hand viewer.

## Configuration overview

The built-in settings menu groups options into:
- **Combat HUD** for incoming damage, expected HP, multi-hit totals, and turn summary tracking.
- **Multiplayer** for teammate hand viewer behavior and sizing.
- **Gameplay** for instant fast mode, splash skipping, and the clock.
- **Party** for per-player map drawing visibility in multiplayer runs.
